Genre Mash-Up


Core Statement

Genre Mash-Up is a Fighter game where each character is based on a different video game genre. My main goals for this project are to learn how to make fighter games and find ways to make that genre more accessible.



Game Pillars

Reimagined Mechanics: Every characer is made with a specific genre in mind,

Mechanics Design

Early in the project I decided that each fighter should have mechanics tied to their genre inspirations. This was challenging for the Fashion character as I tried to balance the slow methodical process in that genre with the fast pace of fighter games. The solution was an outfit system that allowed the Fashion fighter to dress their opponent.. Each piece of clothing has one attribute: Preppy, Cool, and Cute. The opponent has three areas they can be clothed: Tops, Bottoms, and shoes. Below is a table with the outcomes based on the pieces they’re wearing when the outfit is complete:


Reference

This concept requires a vast amount of references due to the nature of its characters. Each character has 4 in-genre reference points and several fighter game specific references. For example the character Fighter was inspired by Street Fighter 6, King of Fighters, Guilty Gear Strive and Marvel vs. Capcom: Clash of Super Heroes. The character’s moveset has specific character’s attacks as references like Ryu’s Hadoken or Zangief’s Screw Pile Driver. You can find out more about this on Fighter’s Confluence page.

Concept Creation

When I start a new design, I like to establish the Design Pillars that will guide the rest of development. For Genre Mash-Up I decided on these three:






Battle Planning

I wanted to make sure each design not only paid homage to different genres but were intuitive to players coming from that different mindset. An FPS player should be able to pick up Shooter and have some of their experiences carry over.




Keep Fighter Game Fundamentals in Mind

The only time the first Design Pillar should be avoided if it can ruin the fighter game experience. The idea here is to remind everyone on the team that this project will be a constant balancing act between reference and balance.

Design with Casual Players in Mind

I want Genre MashUp to be as accessible as possible. Every design decision in this game should ask “does this help lower the barrier of entry for new players?” This question is what lead to the concept in the first place: making characters that have mechanics pulled from the genres they’re based on to make this game less intimidating for those player bases.


Next Steps

My next milestone for this project is two more characters concepted out based on Sports and Dress-Up games. After that I’ll concept out a stage and some music to go with it before creating a vertical slice prototype.
